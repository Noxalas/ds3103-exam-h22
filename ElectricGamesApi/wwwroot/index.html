ElectricGamesApi

API for ElectricGames

Introduction

Welcome to the official documentation of the ElectricGamesAPI. This page aims at giving you a broad overview over the endpoints available and the provided functionality.

If you want a more detailed look at the schema design and the available endpoints, please use Swagger.

Schemas

Game

Data points:

  • id
    • int PK, game identifier.
  • title
    • string, game title.
  • platform
    • string, name of the game's platform.
  • releaseYear
    • int, year of release.
  • image
    • string, image name (including extension)

Character

Data points:

  • id
    • int PK, character identifier.
  • name
    • string, character name.
  • gameId
    • int FK, id of the character's game of origin.
  • game
    • Game, the game's object.

End Points

Ability

GET/api/Ability

Returns a List<Ability> of all Abilities

GET/api/Ability/{id}

Returns an Ability by the given id

DELETE/api/Ability/{id}

Deletes an Ability by the given id

POST/api/Ability/{ability}

Posts a new Ability to the database using data from passed object

PUT/api/Ability/{ability}

Updates existing Ability in the database using data from passed object

Character

GET/api/Character

Returns a List<GameCharacter> of all characters

GET/api/Character/{id}

Returns a GameCharacter by the given id

DELETE/api/Character/{id}

Deletes a GameCharacter by the given id

POST/api/Character/Post/{gameCharacter}

Posts a new GameCharacter to the database using data from the passed object

PUT/api/Character/{gameCharacter}

Updates an existing GameCharacter in the database using data from passed object

Game

GET/api/Game

Returns a List<Game> of all games

GET/api/Game/{id}

Returns a Game by the provided id

DELETE/api/Game/{id}

Deletes a Game by the given id

GET/api/Game/GetByTitle/{title}

Returns a Game by the passed {title} parameter

POST/api/Game/{game}

Posts a new Game to the database using data from a passed object

PUT/api/Game/{game}

Updates an existing Game entry in the database using data from a passed object

Quiz

GET/api/Quiz

Returns a List<Quiz> of all quizzes

GET/api/Quiz/{id}

Returns a Quiz by the passed id

DELETE/api/Quiz/{id}

Deletes a Quiz with the corresponding id

POST/api/Quiz/{quiz}

Posts a new Quiz to the database.

PUT/api/Quiz/{quiz}

Updates an existing Quiz using data from the passed object

QuizQuestion

Returns a List<QuizQuestion> of all questions

GET/api/QuizQuestion

Returns a QuizQuestion by the passed id

GET/api/QuizQuestion/{id}

Deletes a QuizQuestion with the corresponding id

DELETE/api/QuizQuestion/{id}

Deletes a QuizQuestion with the corresponding id

GET/api/QuizQuestion/GetQuestionsByQuizId/{quizId}

Returns a List<QuizQuestion by the passed id. Best way to find all questions related to a quiz.

POST/api/QuizQuestion/{question}

Posts a new QuizQuestion to the database.

PUT/api/QuizQuestion/{question}

Updates an existing QuizQuestionusing data from the passed object

QuizQuestionAnswer

GET/api/QuizQuestionAnswer

Returns a List<QuizQuestionAnswer> of all answers

GET/api/QuizQuestionAnswer/{id}

Returns a QuizQuestionAnswer by the passed id

DELETE/api/QuizQuestionAnswer/{id}

Deletes a QuizQuestionAnswer with the corresponding id

GET/api/QuizQuestionAnswer/GetAnswersByQuestionId/{questionId}

Returns a List<QuizQuestionAnswer> by the passed id. Use together with a question's id for the best way of pairing them up

POST/api/QuizQuestionAnswer/{questionAnswer}

Posts a new QuizQuestionAnswer to the database.

PUT/api/QuizQuestionAnswer/{questionAnswer}

Updates an existing QuizQuestionAnswerusing data from the passed object

UploadImage

GET/api/UploadImage Returns a string informing the user the endpoint was used correctly.

POST/api/UploadImage Uploads an image file to wwwroot

Unused Schemas

Ability

Data points:

  • id
    • int PK, ability identifier.
  • name
    • string, ability name
  • iconImagePath
    • string, icon path

Quiz

Data points:

  • id
    • int PK, quiz identifier.
  • name
    • string, quiz name.
  • quizText
    • string, the accompanying quiz description.
  • gameId
    • int FK, game ID of the game this quiz is based off of.
  • gameObj
    • Game, game this quiz is based off of.

QuizQuestion

Data points:

  • id
    • int PK, question identifier.
  • question
    • string, question text.
  • multipleAnswersCount
    • int, the count of correct answers this question is related to.
  • quizId
    • int FK, quiz foreign key.
  • quizObj
    • Quiz, quiz object reference.

QuizQuestionAnswer

Data points:

  • id
    • int PK, answer identifier.
  • quizAnswerText
    • string, the answer text.
  • correctAnswer
    • bool, true/false if this is the correct answer.
  • quizQuestionId
    • int FK, question foreign key that this answer is related to.
  • quizQuestionObj
    • QuizQuestion, question object reference.
